# 🏫 Aqbobek AI - Интеллектуальный помощник директора школы

Инновационная система управления школой на базе ИИ (Google Gemini 1.5 Flash), разработанная для автоматизации рутины директора, маршрутизации задач и интеллектуального управления расписанием.

## 🚀 Основные возможности

- **🧠 Командный центр (AI Assistant):** Распознавание команд на естественном языке. Директор может продиктовать или написать запрос (например, *"Замени учителя Иванову на сегодня"*), и ИИ автоматически определит намерение, извлечет данные и предложит действия.
- **📅 Продвинутое расписание:**
  - Интеграция реальных данных из PDF-файлов.
  - Два режима: детальный (по классам) и общий вид (Excel-сетка всей школы).
  - Интеллектуальная обработка "окон" и перерывов на питание.
  - Возможность масштабирования (Zoom) для обзора всей школы (до 20 параллелей одновременно).
- **🎙️ Голосовой ввод:** Интеграция Voice-to-Text через собственный backend. Возможность диктовать команды в чат и редактировать расписание голосом.
- **🛠️ Service Desk:** Автоматическая классификация проблем (IT, Хозчасть, Медпункт) и маршрутизация задач ответственным сотрудникам.

---

## 🛠️ Технологический стек

- **Frontend:** React, Vite, Tailwind CSS, Lucide Icons, Framer Motion.
- **AI:** Google Gemini 1.5 Flash API (Анализ текста + Транскрибация аудио).
- **Backend:** Python (FastAPI), Uvicorn (Транскрибация и обработка аудио).

---

## 🏗️ Инструкция по запуску

Для полной работы системы необходимо запустить две части: фронтенд и бэкенд (для голоса).

### 1. Фронтенд (React + Vite)
```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev
```
*Интерфейс будет доступен по адресу:* `http://localhost:5173`

### 2. Бэкенд (Python - Голосовой ввод)
```bash
# Установка нужных библиотек
py -m pip install fastapi uvicorn google-generativeai python-multipart

# Запуск сервера транскрибации
py server/main.py
```
*Backend запустится на:* `http://localhost:8000` (необходим для работы микрофона).

## 📁 Структура проекта

- `/src/app/components` — основные блоки UI (Расписание, Командный центр, Голос).
- `/src/app/services` — работа с Gemini API.
- `/src/app/data` — обработанные данные из PDF-расписания.
- `/server` — Python бэкенд для обработки звука.
- `/assetsfromais` — исходные PDF материалы расписания.
